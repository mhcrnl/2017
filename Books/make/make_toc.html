GNU Make

A Program for Directing Recompilation

GNU make Version 3.79

April 2000

Richard M. Stallman and Roland McGrath

The GNU make utility automatically determines which pieces of a large program need to be recompiled, and issues the commands to recompile them.

This edition of the GNU Make Manual, last updated 04 April 2000, documents GNU make Version 3.79.
