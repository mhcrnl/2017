About Nautilus

 

Nautilus is an easy to use graphical shell for GNOME that integrates graphical file management, web browsing, system management and more using a sophisticated, extensible component architecture.

Some of its most important features include:
  • An innovative, next generation graphical file manager
  • icons can be any size, and are individually resizable by the user
  • zooming UI shows more detail at higher zoom levels
  • uses file contents (images, text) to represent files
  • user assignable file attributes are represented by graphical emblems
  • a directory can control its own presentation by including a metafile or component
  • content and attribute-based "search folders" frees users from the file hierarchy
  • most documents are viewable directly from the shell via components
  • Multiple user levels offer an optimized UI for both novice and expert users
  • Easy end-user customization via a simple, drag and drop interface
Nautilus has a componentized architecture for unlimited extensibility:
  • component architecture allows new functionality to be added on-the-fly
  • framework is uri-based, like a web browser
  • multiple content views available, based on protocol, MIME type and user preference
  • meta-views to annotate content with descriptive info, and aid navigation
  • extensible, virtual file system allows viewing inside tar archives, packages, etc
  • Mozilla-based component for first-class web browsing
Nautilus is a platform for Internet-based system management services
  • new services can be added at run-time via the component architecture
  • encapsulates software and hardware configuration for service to use
  • file-oriented services (back-up, synchronization)
  • installation and configuration management services
  • software update service
